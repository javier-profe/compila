1.1.- Introducción a la compilación

Caso práctico

Estás escribiendo tu primer programa en C++. Al darle al botón "Ejecutar", el IDE (entorno integrado de desarrollo) te muestra una lista de errores que no entiendes. Una compañera te dice: "Es que el código fuente no es lo que el ordenador lee directamente, primero hay que pasar por el compilador".

¿Qué crees que hace exactamente ese "compilador" con el texto que has escrito?

La compilación es el proceso de traducción de un código fuente (escrito en un lenguaje de alto nivel, como C, Java o C#) a un código de bajo nivel o código máquina que el procesador pueda ejecutar. Sin este proceso, el hardware sería incapaz de interpretar nuestras instrucciones.

  • Editor de texto
  • Código fuente
  • Compilador
  • Código objeto/máquina

Reflexiona

¿Sería posible que un ordenador ejecutara directamente un archivo .cpp o .java sin ningún software de por medio? Piensa en la diferencia entre el lenguaje humano y el binario.

Todo el mundo en este país debería aprender a programar un ordenador, porque eso te enseña a pensar.

Steve Jobs
