1.4.- Herramientas y errores comunes

Para compilar en entornos reales de FP, solemos usar herramientas como gcc (para C), javac (para Java) o los botones integrados en IDEs como Visual Studio Code o Eclipse. Es vital leer el output de la consola para depurar.

Recomendación

Te recomendamos que siempre compiles tu código de forma incremental. No escribas 200 líneas de golpe; compila cada vez que añadas una funcionalidad nueva para localizar errores fácilmente.

Debes conocer

Linker (Enlazador): es la herramienta que une tu código compilado con las librerías del sistema para crear el ejecutable final. Sin el enlazador, el proceso de compilación estaría incompleto.
