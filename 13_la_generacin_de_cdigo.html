1.3.- La generación de código

Una vez que el código es correcto, el compilador genera un código intermedio o directamente el código objeto. Aquí entran en juego los Optimizadores, que intentan que el programa resultante sea lo más rápido y ligero posible.



Para saber más

Existen lenguajes que no se compilan "del todo" antes de ejecutarse, sino que usan una máquina virtual (como Java) o son interpretados (como Python). Investiga el concepto de Just-In-Time compilation (JIT).
